Why We Made This:

TIDE has a lot of different websites that we use as students and it can be difficult to remember and find the links to all of them. We created this website as a hub of all the different links in order to make navigating around easier.

Features:

The order of the links defaults to A-Z, but as you use the website and click the links, it will automatically reorder based on which links you click on most. It can always be reset using the "Reset Link Order" button in the sidebar.

Also in the sidebar is a link to the feedback form if you wish to give any feedback.

There is also a search function at the top of the website where you can put in the website name or terms associated with the website to find it.

Along the top there are also common categories that you can sort by, such as Foothill that has all the links to relevant Foothill websites.

And along the side there are informational resources.
